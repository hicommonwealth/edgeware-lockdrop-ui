Edgeware Lockdrop
If you're new to the Edgeware Lockdrop, visit this page to read more about it and more instructions for participating. If you need an EDG wallet, see getting started.
Lockdrop contract address
Contract audited by Quantstamp. See the audit report, and verify the contract code on Etherscan.
EDG address
Your EDG will go to this address. Verify this is correct.
Selection
Validate at network launch
Select yes to stake your EDG at network launch. This means you must have an online validator node when the genesis block launches, or your balance may be slashed.
Submit transaction
Using a Ledger, Trezor, or Safe-T hardware wallet, or an offline keystore file? Select Metamask or MyCrypto to continue.

You should be prompted by Metamask to sign and send the transaction.
