Edgeware Lockdrop
There are two methods of generating a keypair - using Rust on the command line or through the Commonwealth UI.
Select keypair generation method
Select an option to see instructions for generating keypairs using the selected method.
  • Go to the Commonwealth UI
  • Click "Create Account" in the top right corner
  • Enter an email address or use a Github account to login
  • Save the account mnemonic somewhere safe as the UI instructs
  • Copy the Base58 generated public address for use on the Edgeware Lockdrop homepage
  • Install rust and cargo by running the following script in a command shell 
    curl https://sh.rustup.rs -sSf | sh
  • Execute the following script once cargo is installed 
    cargo install --force --git https://github.com/paritytech/substrate subkey
  • Create an account by running the following command 
    subkey generate
    Save the output somewhere safe, it contains your secret mnemonic to generate future keypairs. It should look like the following. 
    Phrase `welcome draft actor now search injury enhance cigar misery move december essay` is account:
  Seed: 0xa2cc1ebe0a6cb0ae78311cd37156aac988e61c312d73cafe357cc2f1fd069e09
  Public key (hex): 0xb2d1bda2d363cf25a26a9fb34ea314dc8981e378ca4797c52117f65731602826
  Address (SS58): 5G7AgnBBNvdDYRVvcCYz95pMjGtS3ouWaq23Dcqe6rDjDuh7
  • Copy the Base58 generated public address for use on the Edgeware Lockdrop homepage
